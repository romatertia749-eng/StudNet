# Объяснение параметров .env файла

## TELEGRAM_BOT_TOKEN
**Что это:** Токен твоего Telegram бота от @BotFather

**Где взять:**
1. Открой Telegram, найди `@BotFather`
2. Команда `/newbot` (если еще нет бота)
3. Сохрани токен, который даст BotFather
4. Вставь в `.env`: `TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklmnoPQRstuvwxyzABC`

**Зачем:** Нужен для валидации данных от Telegram Mini App. Без него валидация не будет работать (но моковые данные для разработки все равно работают).

---

## JWT_SECRET
**Что это:** Секретный ключ для подписи JWT токенов

**Где взять:** Это просто случайная строка. Можно:
1. Сгенерировать автоматически (я уже сгенерировал для тебя)
2. Или придумать самому (минимум 32 символа, лучше больше)

**Пример сгенерированного:**
```
JWT_SECRET=jTOAbIQpMlvh_oKIVSAdYwIytGynkyo_6dtCLPLHLy0
```

**Важно:** 
- В продакшене используй другой, более сложный ключ
- Никогда не коммить этот файл в git (он уже в .gitignore)
- Если кто-то узнает этот ключ, он сможет подделать токены

**Зачем:** Используется для создания и проверки JWT токенов пользователей после валидации Telegram данных.

---

## DATABASE_URL
**Что это:** Строка подключения к PostgreSQL базе данных

**Формат:**
```
postgresql://username:password@host:port/database_name
```

**Твой случай:**
```
postgresql://postgres:!Miha596169@localhost:5432/networking_app
```

Расшифровка:
- `postgres` - имя пользователя БД
- `!Miha596169` - пароль
- `localhost:5432` - хост и порт (локально)
- `networking_app` - имя базы данных

**Менять нужно только если:**
- Изменился пароль БД
- База на другом сервере
- Другое имя базы

---

## UPLOAD_DIR
**Что это:** Папка, куда сохраняются загруженные фото профилей

**Текущее значение:** `./uploads/photos`

**Что означает:** 
- `./` - относительно папки `backend_python`
- Создастся автоматически при первой загрузке

**Менять нужно только если:** Хочешь сохранять фото в другое место

---

## MAX_FILE_SIZE
**Что это:** Максимальный размер загружаемого файла в байтах

**Текущее значение:** `5242880` (это 5 МБ)

**Расчет:**
- 1 МБ = 1024 * 1024 = 1,048,576 байт
- 5 МБ = 5 * 1,048,576 = 5,242,880 байт

**Менять нужно только если:** Хочешь разрешить файлы больше/меньше 5 МБ

## Быстрая настройка

Просто вставь в `.env` файл сгенерированный JWT_SECRET:

```
JWT_SECRET=jTOAbIQpMlvh_oKIVSAdYwIytGynkyo_6dtCLPLHLy0
```

Или сгенерируй новый:
```bash
python -c "import secrets; print(secrets.token_urlsafe(32))"
```
