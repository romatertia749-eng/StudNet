# Настройка Cloudinary для хранения фотографий

## Что такое Cloudinary?
Cloudinary — облачное хранилище для изображений. Бесплатный план включает:
- 25 GB хранилища
- 25 GB трафика в месяц
- Автоматическая оптимизация изображений
- Постоянные URL для фотографий

## Шаг 1: Регистрация на Cloudinary
1. Перейдите на [cloudinary.com](https://cloudinary.com)
2. Нажмите **Sign Up for Free**
3. Зарегистрируйтесь (можно через Google/GitHub)
4. Подтвердите email

## Шаг 2: Получение учетных данных
После регистрации вы попадёте в Dashboard:
1. На главной странице Dashboard найдите **Account Details**
2. Скопируйте следующие значения:
   - **Cloud name** (например: `d123456789`)
   - **API Key** (например: `123456789012345`)
   - **API Secret** (например: `abcdefghijklmnopqrstuvwxyz123456`)

## Шаг 3: Добавление переменных в Koyeb
1. Koyeb → ваш сервис → **Settings** → **Environment Variables**
2. Добавьте три переменные:
   - `CLOUDINARY_CLOUD_NAME` = ваш Cloud name
   - `CLOUDINARY_API_KEY` = ваш API Key
   - `CLOUDINARY_API_SECRET` = ваш API Secret
3. Сохраните и перезапустите сервис

## Шаг 4: Перезапуск бэкенда
После добавления переменных:
1. Koyeb → **Deployments** → **Restart**
2. Или сделайте новый push в GitHub (автоматический деплой)

## Готово!
Теперь фотографии будут сохраняться в Cloudinary и доступны по постоянным URL.

## Проверка
После настройки попробуйте загрузить фото в профиль. Оно должно:
- Успешно загрузиться
- Отображаться в профиле
- Сохраняться при обновлении профиля

## Просмотр всех фотографий
Чтобы посмотреть все загруженные фотографии пользователей:
1. Войдите в [Cloudinary Dashboard](https://cloudinary.com/console)
2. В левом меню выберите **Media Library**
3. Откройте папку **telegram_networking_app** (или используйте поиск)
4. Здесь будут все фотографии, загруженные через приложение

Вы можете:
- Просматривать фотографии
- Скачивать их
- Удалять при необходимости
- Видеть URL каждой фотографии
