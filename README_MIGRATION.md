# Миграция на облачные сервисы

Проект готов к деплою на:
- **Koyeb** (бэкенд)
- **Vercel** (фронтенд)  
- **Neon** (база данных)

## Быстрый старт

Смотрите `QUICK_MIGRATION.md` для пошаговой инструкции.

- `DEPLOY_CHECKLIST.md` - чеклист для проверки всех шагов

## Структура деплоя

```
┌─────────────┐
│   Vercel    │ → React Frontend
│  (Frontend) │
└──────┬──────┘
       │
       │ API calls
       │
┌──────▼──────┐
│   Koyeb     │ → FastAPI Backend
│  (Backend)  │
└──────┬──────┘
       │
       │ SQL queries
       │
┌──────▼──────┐
│    Neon     │ → PostgreSQL Database
│  (Database) │
└─────────────┘
```
